# 클럽스코어 기록 Feature - 세션 저장

> 저장일시: 2026-01-28
> 상태: 브레인스토밍 완료, 구현 계획 수립 완료

## 세션 요약

**목표**: 골프 라운드에서 각 샷에 사용한 클럽을 기록하고 통계 제공

## 확정된 요구사항

| 항목 | 결정 |
|------|------|
| 기본 입력 모드 | 일반 입력 (클럽은 토글 전환) |
| 기존 스코어 처리 | 추후 클럽 정보 추가 가능 |
| 0퍼트 지원 | 칩인 등 퍼팅 없이 홀인 케이스 |
| 홀별 클럽 표시 | 스코어 테이블과 별도 컴포넌트 (홀번호/클럽/스코어 3열) |
| 전체 클럽 통계 | Stats 페이지 하단에 추가 |
| Supabase | MCP 미연결, SQL 파일 제공 방식 |

## 퍼팅 통계 요구사항

- **라운드별**: 전체 퍼팅 수 (예: 32퍼트)
- **전체 통계**: 라운드당 평균 전체 퍼팅 수 (예: 31.5퍼트/라운드)
- **분포**: 0퍼트/1퍼트/2퍼트/3퍼트/4퍼트+ 개수

## 신규 데이터베이스 테이블

1. `clubs` - 마스터 클럽 목록 (26개)
2. `user_clubs` - 사용자별 보유 클럽
3. `club_shots` - 홀별 클럽 사용 기록 (is_putt 플래그)

## 구현 Phase

1. **Phase 1**: 데이터베이스 (테이블, 시드, RPC)
2. **Phase 2**: 클럽 관리 (Settings)
3. **Phase 3**: 클럽 스코어 입력
4. **Phase 4**: History Detail 클럽 표시
5. **Phase 5**: 전체 클럽 통계 (Stats)
6. **Phase 6**: 마무리 및 엣지 케이스
